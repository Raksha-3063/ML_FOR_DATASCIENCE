# Netflix Dataset Analysis using Pandas Profiling

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) of the Netflix Movies and TV Shows dataset using Python, Pandas, and YData Profiling. The primary objective is to understand the dataset structure, identify data quality issues, handle missing values, and generate an automated profiling report for deeper insights.

## Dataset Information

The dataset contains information about Netflix content, including movies and TV shows. Features include title, director, cast, country, date added, release year, rating, duration, genre, and description.

## Tasks Performed

* Imported and explored the Netflix dataset using Pandas.
* Examined dataset dimensions, columns, and data types.
* Identified missing values across multiple columns.
* Handled missing values using appropriate techniques:

  * Replaced missing categorical values with "Unknown".
  * Filled missing ratings, durations, and dates using the mode.
* Generated a comprehensive profiling report using YData Profiling.
* Analyzed dataset statistics, distributions, missing values, and overall data quality.

## Tools and Libraries Used

* Python
* Pandas
* YData Profiling
* Jupyter Notebook

## Key Outcomes

* Successfully cleaned and preprocessed the dataset.
* Eliminated all missing values.
* Generated an automated EDA report for data exploration.
* Gained insights into Netflix content distribution and dataset characteristics.

## Conclusion

This project demonstrates the fundamental steps involved in data preprocessing and exploratory data analysis. It highlights the importance of handling missing values and using automated profiling tools to better understand a dataset before applying advanced analytics or machine learning techniques.
