# USA Vehicle Insights 2021 - EDA Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the USA Vehicle Insights 2021 dataset using Python.

The analysis includes:
- Data Understanding
- Missing Value Analysis
- Univariate Analysis
- Distribution Analysis
- Outlier Detection
- Data Visualization

---

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- KaggleHub

---

## Dataset Source
Kaggle Dataset:
https://www.kaggle.com/datasets/shivd24coder/usa-vehicle-insights-2021

## Analysis Performed

### Basic Data Analysis
- Dataset shape
- Column analysis
- Data types
- Missing values
- Duplicate checking

### Univariate Analysis
- Histogram
- KDE Plot
- Boxplot
- Countplot

### Bivariate Analysis
- Scatterplot
- Correlation Heatmap

---

## Key Insights
- `MILESANNL` shows a highly right-skewed distribution.
- Presence of outliers in annual mileage data.
- Large number of categorical variables in dataset.
- Dataset contains both numerical and categorical features suitable for advanced analytics.
